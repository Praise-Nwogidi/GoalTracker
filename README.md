GoalTracker
Turn Goals Into Achievements

GoalTracker is a modern, distraction-free web application designed to help users set meaningful goals, track progress visually, build daily streaks, and stay motivated through clean design and smart feedback.

This project was built as a frontend MVP using pure HTML, CSS, and JavaScript, focusing on usability, performance, and modern UI principles.
